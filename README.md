# Healthcare Resource Portal (PWA)

A lightweight, installable Progressive Web App (PWA) designed to provide fast access to essential healthcare resources. This app functions offline, installs like a native app, and organizes quick links, directories, forms, and help desk information into one simple, accessible tool.
